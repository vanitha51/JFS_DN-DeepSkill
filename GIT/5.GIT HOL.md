**Git Hands-on Lab – Pull and Push to Remote Repository**



**Step 1:** Verify if Master is in Clean State

Command

git status

Output

On branch master



Your branch is up to date with 'origin/master'.



nothing to commit, working tree clean

Observation



The master branch is clean, and there are no pending changes in the working directory.

**----------------------------------------------------**

**Step 2:** List All Available Branches

Command

git branch -a

Output

\* master

&#x20; remotes/origin/HEAD -> origin/master

&#x20; remotes/origin/master

Observation

master is the current local branch.

origin/master is the corresponding remote branch.

The \* symbol indicates the active branch.

**----------------------------------------------------**

**Step 3:** Pull the Remote Repository to Master

Command

git pull origin master

Output (If Already Up-to-Date)

From https://github.com/username/repository.git



&#x20;\* branch            master -> FETCH\_HEAD



Already up to date.

Output (If New Changes Exist)

From https://github.com/username/repository.git



Updating 5a6b7c8..9d0e1f2



Fast-forward



2 files changed

Observation



The latest changes from the remote repository were downloaded and merged into the local master branch.

**----------------------------------------------------**

**Step 4:** Verify Git Status After Pull

Command

git status

Output

On branch master



Your branch is up to date with 'origin/master'.



nothing to commit, working tree clean

Observation



The local repository is synchronized with the remote repository.

**----------------------------------------------------**

**Step 5:** Push Pending Changes from "Git-T03-HOL\_002"



Note: Ensure that the pending changes from the previous hands-on (Git-T03-HOL\_002) have already been committed locally before executing the push.



Command

git push origin master

Output

Enumerating objects: 8, done.



Counting objects: 100% (8/8), done.



Writing objects: 100% (5/5), done.



Total 5 (delta 2), reused 0 (delta 0)



To https://github.com/username/repository.git



&#x20;  abc1234..def5678  master -> master

Observation



The committed changes were successfully uploaded to the remote repository.

**----------------------------------------------------**

**Step 6:** Verify Git Status

Command

git status

Output

On branch master



Your branch is up to date with 'origin/master'.



nothing to commit, working tree clean

Observation



The local and remote repositories are fully synchronized.
